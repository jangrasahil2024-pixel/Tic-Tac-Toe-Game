# 🌐 Basic Web Development Projects

This repository contains a collection of beginner-friendly web development projects built using **HTML, CSS, and JavaScript**. These projects were created to improve my frontend development skills and understanding of JavaScript concepts.

## 📂 Projects Included

### 🎮 Tic Tac Toe
- Two-player Tic Tac Toe game
- Interactive game board
- Win and draw detection

### ✂️ Stone Paper Scissors
- Classic Stone-Paper-Scissors game
- Random computer choice
- Score tracking using JavaScript

### 💼 Portfolio Website
- Personal portfolio webpage
- Responsive design
- Sections for About, Skills, Projects, and Contact

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
